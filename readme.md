## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/pedromotta462/EnsinIA.git
```

Entre no diretório do projeto

```bash
  cd ensinia
```
#### Rodando Angular App

Instale as dependências

```bash
  npm install
```

Inicie o servidor de desenvolvimento

```bash
  npx expo start
```
